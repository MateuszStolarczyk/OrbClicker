# OrbClicker

**OrbClicker** to gra typu *clicker* stworzona w ramach projektu szkolnego.

Gra jest open-source.

Zdobywasz punkty przez klikanie i za pomocą ulepszeń zwiększasz tępo progresowania.  
Projekt został wykonany przy użyciu HTML, CSS oraz JavaScript.
#### Logo gry, ładne czy nie, zrobiłem ja.
Dzwięki pobrane z [Pixabay](https://pixabay.com/pl/sound-effects/), a ikonki z [BoxIcons](https://boxicons.com/icons?free=true)!
